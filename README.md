# Russian Roulette

Simple russian roulette game with C++. Your chance is 1 in 6.

## Install

Linux:

```bash
git clone https://github.com/miguelzacca/russian-roulette.git
~/russian-roulette/config
```

Windows:

```cmd
git clone https://github.com/miguelzacca/russian-roulette.git
russian-roulette/config.exe
```

## Use

Linux

```bash
sudo rr
```

Windows:

```cmd
rr
```

## If you lose

Linux:

```bash
sudo shutdown -a now
```

Windows:

```cmd
shutdown /s /f /t 0
```
